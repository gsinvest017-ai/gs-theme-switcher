gs-theme-switcher demo

Three themes: Genesis Gold · VSCode Dark · GS Portal — switch above.

1. Install

<!-- In <head>, BEFORE any other CSS -->
<link rel="stylesheet" href="themes.css">
<script src="theme-switcher.js"></script>

The script applies the saved theme instantly (no flash) and appends the picker to <header> once the DOM is ready.

2. Configuration

<!-- Set BEFORE loading theme-switcher.js -->
<script>
  window.GsThemeSwitcherConfig = {
    lsKey:          'my-app-theme',   // localStorage key
    defaultTheme:   'vscode-dark',    // fallback theme
    headerSelector: '.site-header',  // where to inject picker
  };
</script>
<script src="theme-switcher.js"></script>

3. Programmatic API

// Apply a theme
ThemeSwitcher.apply('gs-portal');

// Get current theme
console.log(ThemeSwitcher.current());  // 'genesis-gold'

// Build picker and place it yourself
const picker = ThemeSwitcher.buildPicker();
document.querySelector('.my-nav').appendChild(picker);

// Listen for changes
ThemeSwitcher._onChange = (themeId) => console.log('switched to', themeId);

4. CSS Variable Palette

All three themes expose the same CSS custom properties so components are theme-agnostic.
